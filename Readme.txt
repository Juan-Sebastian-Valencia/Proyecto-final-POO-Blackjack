README


Alcance y objetivos del proyecto:


Crear un simulador del juego de Blackjack utilizando el lenguaje de programación C++ y el paradigma de programación orientada a objetos (POO). El juego simulará las reglas y una partida convencional de Blackjack, permitiendo al jugador jugar contra el crupier.


Objetivos funcionales:
* Hacer apuestas.
* Plantarse.
* Doblar.
* Dividir.
* Pedir carta.
* Repartir cartas.
* Salir del juego.
* Simular las reglas del BlackJack.
* Guardar datos de los jugadores.
* Mostrar puntajes anteriores.
* Mostrar el puntaje más alto hasta el momento.


Objetivos no funcionales:
* Código modulable y orientado a objetos.
* Aplicar encapsulamiento, herencia y polimorfismo.
* Utilización de los principios SOLID.
* Utilización del patrón de diseño modelo, vista y controlador.
* Documentación clara en formato CRC.
